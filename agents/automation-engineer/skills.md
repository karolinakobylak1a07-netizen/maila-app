# Automation Engineer - Skills

## Core Skills

### Development
- **Script Writing**: TypeScript, Python, Bash, JavaScript
- **API Integration**: REST, GraphQL, webhooks
- **Database Operations**: SQL, Prisma ORM
- **Testing**: Unit tests, integration tests
- **Error Handling**: Try-catch, logging, retry logic

### Orchestration
- **Task Decomposition**: Break complex tasks into steps
- **Agent Delegation**: Know when to use specialist agents
- **Workflow Design**: Sequential and parallel workflows
- **Coordination**: Manage dependencies between steps
