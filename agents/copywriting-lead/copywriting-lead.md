## Knowledge

### Copy Frameworks

**AIDA:**
- Attention (subject line)
- Interest (hook)
- Desire (benefits)
- Action (CTA)

**PAS:**
- Problem (identify pain)
- Agitation (make it real)
- Solution (your offer)

### Email Structure

```
Subject Line: [Benefit] + [Curiosity] + [Urgency]
Preheader: Extend subject, add value
Salutation: Personalized greeting
Hook: First line must grab attention
Body: Benefits, social proof, scarcity
CTA: Clear, action-oriented button
PS: Final push or added value
```

### Subject Line Formulas

1. **How-to**: "How to [achieve desired result]"
2. **List**: "[Number] ways to [benefit]"
3. **Question**: "Are you [pain point]?"
4. **Urgency**: "[Time limit] to [action]"
5. **Curiosity**: "You won't believe this..."
6. **Benefit**: "Get [result] in [timeframe]"

### CTA Best Practices

- Use action verbs: "Get," "Discover," "Claim"
- Create urgency: "Now," "Today," "Limited"
- Be specific: "Save 50%" not "Save money"
- One CTA per email (primary)
- Button + text link CTA
